Web 1.0

1945

ENIAC
Primera computadora electrónica de propósito general; base del cómputo moderno.

1969

ARPANET
Primera red de conmutación de paquetes; antecedente directo de Internet.

1970s

Desarrollo de protocolos de red
Investigación y protocolos que sentaron las bases de TCP/IP.

1983

TCP/IP adoptado
Estándar que permitió la interoperabilidad entre redes.

1989

Propuesta de la World Wide Web (Tim Berners-Lee)
Idea del hipertexto global para compartir información en CERN.

1990

Primer servidor y navegador
Implementación inicial del servidor web y navegador en CERN.

1993

Mosaic
Navegador gráfico que popularizó la Web entre usuarios generales.

1994-1995

Comercialización (Netscape, Yahoo, Amazon 1995)
Nacen portales y comercio electrónico; la Web se abre al mercado.
